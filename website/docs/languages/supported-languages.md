| Language           |       Version          |
|--------------------|------------------------|
| C++                | -std=c++17             |
| Python             | Version 3              |

## Compilation and Execution Commands

| Language     | Compile / Interpreter                                      | Execution Command          |
|:------------:|------------------------------------------------------------|----------------------------|
| C++17        | `g++ -std=c++17 -Wall -DONLINE_JUDGE=1 -o main main.cpp`   | `./main seed testcase`     |
| Python3      |                                                            | `python3 main.py seed testcase`|
