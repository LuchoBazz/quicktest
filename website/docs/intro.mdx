Quick Test CLI is a tool designed to perform stress tests for competitive programming quickly and easily, allowing you to focus entirely on the contest.

This project was inspired by the following screencasts from [Errichto](https://codeforces.com/profile/Errichto):
- [1v1 Coding | 2020 Lockout Championship](https://www.youtube.com/watch?v=uABbBGtEWks&t=1434s)
- [How to test your solution in Competitive Programming, on Linux?](https://www.youtube.com/watch?v=JXTVOyQpSGM)

### Why use Quick Test CLI?

* **Cross-Platform Support**: Works seamlessly on Windows, Linux, and macOS.
* **Multi-Language Support**: Compatible with a wide range of programming languages including C++, Java, Python, Rust, and Go.
* **User-Friendly Interface**: Simple and intuitive to use, requiring no complex configurations.
* **No Bash Scripts**: Eliminates the need to write and maintain tedious bash scripts for testing.
* **Community Standard**: Designed to become a reliable standard tool for the competitive programming community.
* **Open Source**: A collaborative project that grows and improves through community contributions.
